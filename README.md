## 디렉토리 구조

src  
ㅤ|-- api : api 인스턴트 구현  
ㅤ|-- assets : 이미지 리소스 관리 디렉토리  
ㅤ|-- components : 검색창, 필터, 과목 카드 등 컴포넌트 구현  
ㅤ|-- constants : 컴포넌트, 페이지에서 사용한 상수 정의  
ㅤ|-- hooks : Fetch API 등 커스텀 훅 구현  
ㅤ|-- interface : 컴포넌트, 페이지에서 사용할 인터페이스 정의  
ㅤ|-- pages : 페이지 컴포넌트 구현  
ㅤ|-- style : 글로벌 스타일 및 공통 스타일 정의  
ㅤ|-- types : 커스텀 타입 정의
